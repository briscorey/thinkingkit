## Compounding

Small consistent gains accumulate into extraordinary results over time — in finance, skills, relationships, and knowledge.

[Read the full model →](/models/compounding/)

## Diminishing Returns

Each additional unit of input produces less additional output. The first hour of practice helps more than the hundredth.

[Read the full model →](/models/diminishing-returns/)

## What's the difference?

Compounding describes outputs that grow exponentially — each period's gains build on the last. Diminishing Returns describes inputs with decreasing marginal output — each additional unit of effort produces less result. They're opposing forces that often operate simultaneously in different dimensions.

## When to use each

Use Compounding thinking for investments with long time horizons — knowledge, relationships, capital. Use Diminishing Returns thinking for effort allocation — know when additional input has stopped producing proportional output.

## They work together

These models aren't competitors — they're complements. The best thinkers use both.
